# [91. 最短 Hamilton 路径](https://www.acwing.com/problem/content/93/)

[打卡](https://www.acwing.com/activity/content/problem/content/1011/1/)

给定一个 n 个点的带权无向图，求起点 0 到终点 n-1 重不漏地经过每个点恰好一次。

暴力： n 的全排列，再求路径长度。n！\* n。复杂度太高不可取。

状态表示：f[i, j]所有从 0->j 中间走过点存在 i 当中。 i 是一个二进制数，20 位。
属性：最小值
状态计算：

- 枚举倒数第二点分类（最后一个点是 j）。`0 ->...-> k -> j`
- `0 ->...-> k` 的最小值 : `f[i - {j}, k]`
- `f[i, j] = f[i - {j}, k] + a[k, j]`

```c++
#include <cstring>
#include <iostream>

using namespace std;

const int N = 20, M = 1 << N;

int n;
int w[N][N];
int f[M][N];

int main(){
    cin >> n;
    for(int i = 0;i < n; i ++ ){
        for(int j = 0; j < n; j ++ ){
            cin >> w[i][j];
        }
    }

    memset(f, 0x3f, sizeof f);
    f[1][0] = 0;

    for(int i = 0; i < 1 << n; i ++ ){
        for(int j = 0; j < n; j ++ ){
            if(i >> j & 1){
                for(int k = 0; k < n; k ++){
                    if( i - (1 << j) >> k & 1)
                        f[i][j] = min(f[i][j], f[i - (1 << j)][k] + w[k][j]);
                }
            }
        }
    }
    cout << f[(1 << n) - 1][n-1] << endl;
    return 0;
}
```
