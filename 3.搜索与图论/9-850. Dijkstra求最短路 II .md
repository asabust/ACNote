## 堆优化 dijkstra 算法

> 适用于稀疏图用邻接表来存。$O(ElogV)$
>
> c++用优先队列会有冗余。总共的元素个数是$E$。
> 时间复杂度看似变成$O(ElogE)$，但即使$E = V^2$ -> $logE = 2logV$, 况且在稀疏图内$E << V^2$，所以用 priority_queue 并没有改变时间复杂度。

1. 初始化距离 `dist[1] = 0, dist[i] = INT_MAX` 当前已确定最短距离的点保存到`s`中。
2. 遍历不在`s`中每一个点:
   1. 找到一个距离最近的点`t`（用堆来优化 $V^2$ -> $V$）
   2. 并把`t`放到`s`中，
   3. 用`t`更剩余其他点的距离（用堆来更新，$E$ -> $ElogV$）

```c++
#include <cstring>
#include <iostream>
#include <algorithm>
#include <queue>

using namespace std;

typedef pair<int, int> PII;

const int N = 100010;

int n, m;
int h[N], e[N], ne[N], idx;
int dist[N], w[N];
bool st[N];

void add(int a, int b, int c){
    e[idx] = b, w[idx] = c, ne[idx] = h[a], h[a] = idx ++;
}

int dijkstra(){
    memset(dist, 0x3f, sizeof dist);

    priority_queue<PII, vector<PII>, greater<PII>> heap;

    heap.push({0, 1});
    while(heap.size()){
        auto t = heap.top();
        heap.pop();

        int ver = t.second, distance = t.first;
        if(st[ver]) continue;

        for(int i = h[ver]; i != -1; i = ne[i]){
            int j = e[i];
            if(dist[j] > distance + w[i]){
                dist[j] = distance + w[i];
                heap.push({dist[j], j});
            }
        }
    }

    if(dist[n] == 0x3f3f3f3f) return -1;

    return dist[n];
}

int main(){
    cin >> n >> m;

    memset(h, -1, sizeof h);

    while(m --){
        int a, b, c;
        scanf("%d%d%d", &a, &b, &c);
        add(a, b, c);
    }

    cout << dijkstra() << endl;
    return 0;
}
```
