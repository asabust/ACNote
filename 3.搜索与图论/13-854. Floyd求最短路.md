# [854. Floyd 求最短路](https://www.acwing.com/problem/content/856/)

- 基于动态规划，`d[k, i, j]`表示从`i`开始，只经过`1->k`点到`j`点的最短距离。状态更新`d[k,i,j] = d[k-1,i,k] + d[k-1,k,j]`
- 不能处理负权回路的图。因为不存在最短路鸭

```c++
for k :1->n
    for i:1->n
        for j:1->n
            d[i, j] = min(d[i,j], d[i, k] + d[k, j])
```

```c++
#include <cstring>
#include <iostream>
#include <algorithm>

using namespace std;

const int N = 210, INF = 1e9;

int n, m, k;
int d[N][N];

void floyd(){
    for(int k = 1; k <= n; k ++)
        for(int i = 1; i <= n; i++)
            for(int j = 1; j <= n; j++)
                d[i][j] = min(d[i][j], d[i][k] + d[k][j]);
}

int main(){
    cin >> n >> m >> k;

    for(int i = 1; i <= n; i ++){
        for(int j = 1; j <= n; j ++){
            if(i == j) d[i][j] = 0;
            else d[i][j] = INF;
        }
    }

    while(m--){
        int a, b, c;
        cin >> a >> b >> c;
        d[a][b] = min(d[a][b], c);
    }

    floyd();

    while(k--){
        int i, j;
        cin >> i >> j;
        if(d[i][j] > INF/2) puts("impossible");
        else cout << d[i][j] << endl;
    }
    return 0;
}
```
